# My Kitchen and Bath — Portfolio Version

This repository is a **portfolio-safe archive** of a website project developed for **My Kitchen and Bath**.

## Project overview
This project was created as a **marketing and SEO-focused corporate website** for a remodeling and home improvement business. The site structure was designed to support:
- service promotion
- product/category presentation
- project showcase pages
- blog/content publishing
- contact and lead generation flows
- administrative content management

## What this portfolio version includes
- public-facing ASP.NET Web Forms pages
- master page and layout structure
- selected service, product, blog, project, and contact pages
- admin panel pages that demonstrate content management workflows
- essential assets needed to understand the project structure

## Additional work completed for the company
Beyond this archived website, the engagement also included work on broader internal/business-side solutions such as:
- agenda and scheduling related workflows
- staff / employee follow-up features
- project tracking and operational management work
- business process improvements around website and panel usage

Some of those internal tools are **not included in this portfolio package**, either because they belong to a different codebase, were implemented later, or were intentionally excluded for privacy and security reasons.

## Important note
This is **not the original production package**. To make the project safe to share publicly:
- passwords, connection strings, and sensitive configuration values were removed
- environment-specific and local IDE files were excluded
- non-essential files were removed
- some files may be incomplete or outdated because this is an archived version recovered after the original site lifecycle

## Tech stack
- ASP.NET Web Forms
- C#
- SQL Server
- HTML / CSS / JavaScript

## Purpose of this repository
This repository is intended to **showcase the scope of work and project architecture** rather than serve as a production-ready deployment package.
